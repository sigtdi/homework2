Подзадача 4: # Павлова
Зависимость времени работы функций от размера графа отображена на графике "Время работы функций"

Определение асимптотического наихудшего времени работы алгоритмов targeted_order и fast_targeted_order в зависимости от n

Заметим, что если в графе n вершин, то ребер меньше чем 5n, поскольку параметр m UPA графа равен 5

targeted_order:

1. Создание копии графа: T(n) ≈ n
2. Удаление вершины: T(n) ≈ m, где m - параметр UPA графа, m = 5
3. Поиск порядка удаления вершин: T(n) ≈ n * (n + 5) - n раз ищем за n вершину с максимальной степенью и за m = 5 удаляем

Итог: T(n) ≈ 6 * n + n^2 ≈ O(n^2)

fast_targeted_order:

1. Создание копии графа: T(n) ≈ n
2. Удаление вершины: T(n) ≈ m, где m - параметр UPA графа, m = 5
3. Создание списка степеней вершин: T(n) ≈ n
3. Поиск порядка удаления вершин: T(n) ≈ n * (m + m) - для каждой из n вершин меняем значения в списке степеней вершин для её соседей за m и за m удаляем вершину из графа

Итог: T(n) ≈ n + n + 10n ≈ O(n)
